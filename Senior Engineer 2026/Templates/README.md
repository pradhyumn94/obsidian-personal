# Templates

## What belongs here
The five reusable note templates for this vault: `HLD Template.md`, `LLD Template.md`, `Technical Topic Template.md`, `DSA Problem Template.md`, `Weekly Review Template.md`.

## How to use it
Since this vault deliberately avoids plugins/automation, there's no auto-insert:
1. Open the relevant template file.
2. Select all, copy.
3. Create your new note in the right folder, paste, then fill in the `{{placeholders}}` and delete them.

(If you later want one-click template insertion, Obsidian's built-in core **Templates** plugin — not a community plugin — can point at this folder. Optional, not required to use this vault.)

## Naming convention
Fixed names, one template per note type. Don't create new templates unless a genuinely new note type shows up more than once or twice.
